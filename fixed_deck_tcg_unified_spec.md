# 固定順デッキ型TCG（ブラウザプロトタイプ）統合仕様書

> 世界観：星典×星霊（暫定）
> インスピレーション元：「金色のガッシュベル!! THE CARD BATTLE」の固定順デッキメカニクス
> 独自要素：共鳴（レゾナンス）・過詠（オーバーチャージ）

---

# 1. 概要（企画）

## ■ コンセプト
運要素を極力排除し、「構築」と「読み合い」に特化したカードゲームをブラウザ上で実現する。

プレイヤーは**星詠み師**として、星の力を記した**「星典」**を持ち、
星を詠むほど力を引き出せるが、星典の星が消えれば術者の運命も尽きる。

## ■ コア体験

### 星典のジレンマ
> **星典 = デッキ = ライフ** の三位一体
> - 星を詠めばSP（リソース）を得られる
> - しかし星典が尽きれば敗北する
> - 「今詠むか、温存するか」が常に勝敗を左右する

### 読み合いの攻防
> - 攻撃側：どの星霊 × どの攻星術で仕掛けるか（共鳴を狙うか？）
> - 防御側：守星術で防ぐか、星霊で星護するか、星典で受けるか
> - 過詠で一気に強化するか、SPを温存するか

## ■ 独自システム
> - **共鳴（レゾナンス）**：星霊と星術の属性一致で +1 ボーナス
> - **過詠（オーバーチャージ）**：追加SPで術を強化するリスク/リターン

## ■ ターゲット
- 思考戦・戦略ゲームが好きなユーザー
- 将棋・チェス・Slay the Spire系が好きな層

# 2. ゲーム仕様

## ■ ゲーム形式
- 1対1（プレイヤー vs CPU）
- ターン制
- ローカル環境で完結

## ■ 勝利条件（2つ — いずれかで勝利）
1. **星典消滅**：相手の星典ページが0になる
2. **星霊全滅**：相手の場に星霊がいなくなり再召喚不可

## ■ 星典（デッキ）仕様
- 枚数：20ページ（MVP）
- 順番固定（シャッフルなし）
- 1ページ目は必ず星霊カード
- 星典は「デッキ」「リソース生産源」「ライフ」の三役

## ■ カード種類

### 1. 星霊カード
- フィールドに配置する戦闘の本体
- 星力値（power）と属性を持つ
- 状態管理：輝態 → 蝕態 → 消星
- 星護（かばう）が可能

### 2. 星術カード
- 攻星術（攻）：自分のバトルフェイズで使用
- 守星術（防）：相手のバトルフェイズで使用
- 星力上昇値 + ダメージ値 + 属性を持つ
- 過詠で強化可能

### 3. 星命カード
- 即時効果（1ターン1枚制限）
- 回復・バフ等

## ■ リソース（SP：星力）
- 星典を1ページ詠むごとに **SP+2**
- スタートフェイズで0〜3ページ任意に詠める
- エンドフェイズで1ページ自動詠み
- 未使用SPは持ち越し可能

# 3. ゲームフロー

## ■ ゲーム開始
1. 星典をセット（20ページ、順番固定）
2. 1ページ目の星霊を場に召喚
3. 星典を1ページ詠む → SP+2
4. 先攻/後攻決定

## ■ ターン処理
```
スタートフェイズ
  └─ 星典を0〜3ページ任意に詠む（SP獲得）

メインフェイズ
  ├─ 天窓（見開き）のカードを使用
  ├─ 星霊を場に召喚
  └─ 星命を使用（1ターン1枚）

バトルフェイズ
  ├─ 星撃宣言
  ├─ 攻星術使用 + 過詠（攻撃側）
  ├─ 守星術 + 過詠（防御側）or パス
  ├─ 星力衝突（共鳴ボーナス反映）
  └─ ダメージ解決（星典めくり or 星護）

エンドフェイズ
  ├─ 場の整理（消星した星霊除去）
  ├─ 星典1ページ自動詠み → SP+2
  └─ 勝敗判定
```

## ■ 星力衝突
```
攻撃側合計 = 星霊の星力値 + 星術の星力上昇値 + 共鳴ボーナス + 過詠ボーナス
防御側合計 = 星霊の星力値 + 守星術の星力上昇値 + 共鳴ボーナス + 過詠ボーナス（未使用なら0）

攻撃側 > 防御側 → 攻撃成功（同値は防御有利）
```

## ■ 独自システム：共鳴（レゾナンス）
- 星霊と星術の属性一致で **+1** ボーナス
- 構築段階での「属性合わせ」が戦略の鍵

## ■ 独自システム：過詠（オーバーチャージ）
- 星術のコスト + 追加SP（上限：コスト×2）
- 追加SP 2ごとにダメージ+1 or 星力上昇+1
- SPを多く使う = 星典を多く詠む = リスク増

## ■ ダメージ解決
- 攻撃成功時：ダメージ + 過詠ボーナス分の処理
- 防御側の選択肢：
  - **星典で受ける** → ダメージ分ページがめくられる
  - **星霊で星護** → 星霊が蝕態（蝕態なら消星）
